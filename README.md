# MindfulAI 🧘

A Computer Vision AI System for Personalised Mindfulness Interventions
Based on Emotional State Recognition.

## Tech Stack
- Frontend: React 19 + Vite + Tailwind CSS
- Emotion Detection: face-api.js (client-side, no data sent to server)
- Backend: Django 5.2 + Django REST Framework
- Database: PostgreSQL 16
- Auth: JWT (SimpleJWT)

## Running the Project

### Backend
cd backend
venv\Scripts\activate
python manage.py runserver

### Frontend
cd frontend
npm run dev

## URLs
- React App: http://localhost:5173
- Django API: http://127.0.0.1:8000/api
- Django Admin: http://127.0.0.1:8000/admin

## Features
- Real-time facial emotion recognition
- Personalised mindfulness activity recommendations
- JWT authentication
- Session history tracking
- Community peer support (coming soon)
